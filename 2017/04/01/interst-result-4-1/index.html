interst result_4_1

2017年4月1日

前几天一个学弟告诉我关于复数域上幂零矩阵$A$的一个充要条件,特此记录
$$A = AB-BA for some B \in M_n{\mathbb{C}}$$

证明分几个小步骤。
必要性：

  1. 若$A$为（上三角）若当块，那么取 $B = \diag {0,1,\cdots,n-1}$ 即可，若 $A$ 为分块若当块（若当标准型），那么取对应的分块 $B$ 即可。又由于
    $$ P^{-1} A P = P^{-1} A P P^{-1} B P -P^{-1} B P P^{-1} A P $$
    因此，由对若当标准型成立即可知道对一般形式成立。
  2. $\tr(A^k) = 0, 1 \leq k \leq n$,则 $A$ 幂零。
    由若当标准型，可知，只需证明
    \begin{cases}
    x_1 + x_2 + \cdots + x_n = 0 \
    x_1 ^ 2 + x_2 ^ 2 + \cdots + x_n ^2 = 0 \
    \vdots \
    x_1 ^ n + x_2 ^n + \cdots + x_n ^n = 0
    \end{cases}
    推出，$x_1 = x_2 = \cdots x_n = 0$， 不妨设
    x_1 t_1 + x_2 t_2 + \cdots + x_r t_r = 0 \
    x_1 ^ 2 t_1 + x_2 ^ 2 t_2 + \cdots + x_r ^2 t_r= 0 \
    \vdots \
    x_1 ^ r + x_2 ^r + \cdots + x_r ^r t_r = 0
    \end{cases}
    其中 $x_i$ 互不相同，那么，假设$r>1$ 则由 Vandemode 行列式不为0知，$t_i = 0$ 矛盾，因此 $r = 1$,因此 $x_i = 0$。
  3. $C = AB-BA$ 且 $AC = CA$ 则，$C$ 幂零。
    $$ \forall k,\tr (C^k) = \tr(C^{k-1}AB - C^{k-1}BA) = \tr(C^{k-1}AB) - \tr(AC^{k-1}B) = 0$$，因此由上面结论知，$C$ 幂零。
  4. 由上面结论，充分性显然。
